多指连点器
多指点击器是一款无障碍辅助工具，专为有需要的用户提供触摸模拟功能。 【主要功能】 ✅ 辅助行动不便用户进行触屏操作 ✅ 协助进行应用功能测试和演示 ✅ 支持自定义点击位置、频率和时长 ✅ 可保存多个操作配置方案 【无障碍权限说明】 本应用使用 AccessibilityService API 仅用于： • 在用户指定位置执行点击操作 • 模拟触摸手势以辅助操作 • 不会访问、收集或传输任何个人数据 • 不会监控其他应用或系统信息 【权限控制】 您可以随时在 设置 > 无障碍功能 中禁用本应用的权限。 禁用后，应用将停止所有触摸模拟功能。 【数据安全】 本应用完全离线运行，不收集用户数据，不连接网络。 所有配置信息仅保存在本地设备中。 适用于需要触摸辅助的用户，请合理合法使用。
